highlights:
--------------------------


	- no functions

	- names can contain spaces

	- only binary literals

	- no compound operators, only   

		+ addition
		- subtraction
		* multiply
		/ division
		% remainder 
		& and
		| or
		~ xor
		^ shift up
		! shift down
		() parenthesis for precedence
		@ load/store
		< set to 1 if less than, else 0




	only certain gramatically correct statements:


		[your comment here]


		define N = E

		set @(E) = E

		set @ N = E

		set N = E


		if C then S else S end

		repeat S while E


		do N

		at N


		eoi
