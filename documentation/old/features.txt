
n is a minimalist procedural and imperative systems programming language.


features of n:
========================

	- has no syntax.

	- has complete control over much of the LLVM IR language.

	- has powerful dependent and computational strong type system.

	- has arbitary compile time computation.

	- has typed hygenic macros.

	- has utf8 unicode character support.









things this language does NOT have:
====================================




	- no comments.

	- no strings.

	- no character literals.

	- no builtin numeric constants.

	- no mathematical operators.

	- no var args.

	- no namespaces.

	- no for loops. 

	- no if statements.

	- no while loops.
